Summary
-------
This PR adds a comprehensive "Blueprint: Integrations & Environment Variables"
article documenting how the site wires external services (Upstash, Turnstile,
Resend, GitHub, Vercel), the env vars involved, failure modes, and security and
operational guidance. It includes runbooks, local dev/test instructions,
observability recommendations, a CI Playwright cron example, accessibility
checks, an env var appendix table, and diagrams illustrating the architecture
and contact integration flow.

What to verify

```bash

PORT=3001 pnpm start
```

- Open the article at `/articles/blueprint-integrations` (or the site's MDX route)
  and verify diagrams render and captions display correctly.
- Run unit tests and Playwright E2E against a running server:

```bash

```

PNG fallbacks (optional)
------------------------
This PR includes a helper script to generate PNG fallbacks for the SVGs. To
generate and include PNGs in the PR (optional):

```bash
chmod +x scripts/generate-diagrams.sh
./scripts/generate-diagrams.sh
git add public/blueprint-assets/png/*.png
git commit -m "docs: add PNG fallbacks for blueprint diagrams"
git push
```

Commands to open PR
-------------------
After committing the branch locally, run:

```bash
git push --set-upstream origin add/blueprint-integrations
gh pr create --title "docs: add blueprint integrations article + diagrams" --body-file .pr-body.md
```
