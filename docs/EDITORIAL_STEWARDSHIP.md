# Editorial Stewardship (Phase 5)

This repository treats writing quality as the primary product.

Before publishing any new investigation, validate all of the following:

1. Inquiry depth
- The piece adds new reasoning, not just summaries.
- Core claims are supported with evidence or explicit assumptions.

2. Ecosystem continuity
- It references at least one prior investigation, concept, or pathway.
- It clarifies what is new relative to existing canon texts.

3. Canon integrity
- If marked `canonical_text: true`, it should define or reshape a foundational idea.
- Canon status should be used sparingly and justified editorially.

4. Temporal relevance
- Add `temporal_callbacks` when an older investigation is extended or revised conceptually.
- Prefer enduring framing over trend-driven framing.

5. Atmospheric restraint
- Remove any section that does not deepen inquiry, coherence, or reflection.
- Favor fewer, stronger connections over dense link clusters.

This checklist is intentionally lightweight and static-first.
