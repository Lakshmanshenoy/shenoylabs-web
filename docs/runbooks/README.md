# Runbooks

This folder contains short operational runbooks for common incidents related to
the site's integrations. Each runbook provides quick checks, verification
commands, mitigation steps, and follow-up actions.

Runbooks included:

- [Upstash KV runbook](./upstash.md)
- [Cloudflare Turnstile runbook](./turnstile.md)
- [Resend (email) runbook](./resend.md)
- [GitHub PR/commit runbook](./github.md)
- [Consent / DSR runbook](./dsr.md)

If you add or update a runbook, please ensure it includes verification commands
that do not leak secrets and a short list of escalation contacts.
